# Chats App

Aplikasi komunikasi antar pengguna secara real-time. Aplikasi dibuat menggunakan Flutter 2 dengan Getx state management. 

## Feature

- GetX State Management
- Firebase Auth
- Firebase Storage
- Firebase Firestore
